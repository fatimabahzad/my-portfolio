2D Space Cowgirl Game

Pixel-style hackathon game built with Python and Pygame

About the Project

Space Cowgirl is a 2D pixel-art game developed during a hackathon. The player controls a cowgirl character navigating a space-western landscape, dodging and catching UFOs while managing a lives system. The game was built entirely from scratch in Python using the Pygame library within the hackathon time limit.

How It Works

The game loop runs at a fixed frame rate, handling player input, sprite movement, collision detection, and score tracking each frame. UFO enemies spawn at randomized positions and speeds, increasing in difficulty over time. The lives system is represented visually with heart icons on screen, and the catch counter tracks the player's score in real time.
